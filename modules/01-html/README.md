# Module 1: HTML Fundamentals

Welcome to **Module 1: HTML Fundamentals**! This module covers the essential building blocks of web development using HTML. Below is a summary of the topics completed and the hands-on exercises accomplished in this module.

## ✅ Topics Covered

- **HTML Anatomy**: Understanding elements, tags, and attributes
- **Essential Tags**: `<html>`, `<head>`, `<title>`, `<body>`
- **Content Tags**: `<h1>`, `<p>`, `<div>`, `<span>`
- **Heading Tags**: `<h1>` to `<h6>`, `<span>`
- **Self-closing Tags**: `<hr>`, `<br>`
- **Attributes & Classes**: `id`, `class`, `style` attributes
- **Hyperlinks**: `<a>` tags, absolute and relative links, email links, anchor links
- **Images**: `<img>` tags, alt text, local and network images
- **Lists**: `<ul>`, `<ol>`, `<li>`, and description lists
- **Tables**: `<table>`, `<tr>`, `<td>`, `<th>`
- **Semantic Elements**: `<nav>`, `<header>`, `<section>`, `<aside>`, `<footer>`

## 📝 Hands-on Exercises

All topics above are reinforced with practical exercises, including:

- Creating a basic HTML page structure
- Using headings and content tags
- Applying self-closing tags for layout
- Styling elements with classes and attributes
- Adding hyperlinks (internal, external, email, and anchor)
- Displaying images from local and network sources
- Organizing content with ordered, unordered, and description lists
- Presenting data in tables
- Structuring pages with semantic elements

## 🎯 Outcome

By completing this module, you have:

- Built a strong foundation in HTML
- Practiced creating and structuring web pages
- Learned to use semantic and structural tags for better accessibility and SEO
- Developed hands-on skills through real HTML files and examples
